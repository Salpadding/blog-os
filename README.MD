# blog os

1. install rust nightly

```sh
rustup add nightly
```

2. install cargo-xbuild

```sh
cargo install cargo-xbuild
```

3. install rust source code

```sh
rustup component add rust-src
```

4. install bootimage

```sh
cargo install bootimage
```

5. add llvm tools

```sh
rustup component add llvm-tools-preview
```
